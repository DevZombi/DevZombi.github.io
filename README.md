# DevZombi.github.io

This project is an effort to do something artistic with my programing skills. I wanted to build something that is not just processing data or configuring a third aprty device. 

I will be supporting this project with a few videos on YouTube (comming soon). And the art generated will be for sell at openseas.

## The Project Phase 1 (100 images will be generated for sale)

The main goal was to generate the squares. They are generated in 3 different sizes. And filled with random material colors.

I chose the material color pallete both because it recalls the memory to apps and software ui, and because they are pleasing soft colors.

Lastly for phase 1 I wanted to also develop a signature to sign the art with. Also generates a bit at random and fills colors in at random.
